Для запуска мобильного приложения:

  1. Скачай Node.js: https://nodejs.org
  2. Установи (выбери LTS версию)
  3. Перезапусти компьютер
  4. Открой терминал в папке mobile
  5. Выполни: npm install
  6. Затем: npx expo start
