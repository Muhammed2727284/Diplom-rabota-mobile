# 📦 Установка Node.js для запуска Frontend

## Проблема
Frontend не запускается, потому что **Node.js не установлен**.

## ✅ Решение (5 минут)

### Шаг 1: Скачай Node.js
Перейди на сайт: **https://nodejs.org/en/download/**

### Шаг 2: Выбери версию
- Нажми на кнопку **"LTS"** (рекомендуется)
- Скачай установщик для Windows (.msi файл)

### Шаг 3: Установи
1. Запусти скачанный файл
2. Нажимай "Next" во всех окнах
3. Согласись с лицензией
4. Дождись завершения установки

### Шаг 4: Перезапусти терминал
**ВАЖНО:** Закрой все терминалы и открой новый

### Шаг 5: Проверь установку
Открой терминал и выполни:
```bash
node --version
npm --version
```

Должны появиться версии (например: v20.11.0 и 10.2.4)

### Шаг 6: Запусти Frontend
```bash
cd mobile
npm install
npx expo start
```

## 🎯 Пока Node.js не установлен

Открой файл: **`backend/test_api.html`** в браузере

Там можно:
- Проверить работу API
- Зарегистрировать пользователя

## ❓ Часто задаваемые вопросы

**Q: Нужно ли платить за Node.js?**
A: Нет, Node.js полностью бесплатный.

**Q: Какую версию выбрать?**
A: Всегда выбирай **LTS** (Long Term Support) - это стабильная версия.

**Q: Сколько места займёт?**
A: Около 50-100 МБ.

**Q: Безопасно ли это?**
A: Да, Node.js - это официальная платформа, используемая миллионами разработчиков.

**Q: После установки всё равно не работает?**
A: Обязательно **перезапусти терминал** или даже компьютер.
