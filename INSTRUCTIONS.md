# 🚀 Запуск JobBoard проекта

## Быстрый запуск (1 клик)

Просто **дважды кликни** на файл:
```
START_PROJECT.bat
```

Откроются 2 окна:
1. **Django Backend** - http://localhost:8000
2. **Expo Frontend** - с QR-кодом

## Подключение с телефона

1. Открой **Expo Go** на телефоне
2. Нажми **"Scan QR code"**
3. Отсканируй QR-код из окна Expo
4. Приложение загрузится!

## ⚠️ Важно для физического телефона

Если телефон не подключается, измени IP в файле:
`mobile/src/api/client.js`

```javascript
// Строка 3: замени на IP твоего компьютера
const API_BASE_URL = 'http://192.168.X.X:8000/api';
```

Узнать IP компьютера:
```bash
ipconfig
```
Найди "IPv4 Address" (например: 192.168.1.100)

## 📱 Тестирование

1. Зарегистрируй аккаунт (выбери роль USER или ORG)
2. Войди в систему
3. Проверь функционал:
   - **USER**: создание резюме, отклики на вакансии
   - **ORG**: создание вакансий, просмотр откликов

## 🛑 Остановка серверов

Закрой оба окна терминала или нажми `Ctrl+C` в каждом.

## ❓ Проблемы?

**Expo не запускается:**
- Закрой все терминалы
- Запусти `START_PROJECT.bat` заново

**Телефон не видит QR-код:**
- Убедись что телефон и компьютер в одной WiFi сети
- Измени IP в `client.js` (см. выше)

**Backend не работает:**
- Проверь что порт 8000 свободен
- Перезапусти `START_PROJECT.bat`
