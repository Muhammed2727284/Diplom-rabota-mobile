  JobBoard - Быстрая Инструкция
========================================

ЗАПУСК ПРОЕКТА:
1. Дважды кликни на файл: INSTALL_AND_RUN.bat
2. Подожди пока установятся зависимости (1-2 минуты)
3. Откроются 2 окна: Backend (синее) и Frontend (желтое)
4. В окне Frontend появится QR-код

ПОДКЛЮЧЕНИЕ С ТЕЛЕФОНА:
1. Открой Expo Go на телефоне
2. Нажми "Scan QR code"
3. Отсканируй QR-код из желтого окна
4. Приложение загрузится!

ЧТО ИСПРАВЛЕНО:
✅ Проект обновлен до SDK 54 (совместим с твоим Expo Go)
✅ API настроен на localhost:8000
✅ Backend запускается автоматически
✅ Frontend запускается с tunnel режимом
✅ Все зависимости обновлены

ЕСЛИ НЕ РАБОТАЕТ:
1. Убедись что Node.js установлен (node --version)
2. Убедись что Python установлен (python --version)
3. Проверь интернет соединение (для tunnel)
4. Перезапусти INSTALL_AND_RUN.bat

ОСТАНОВКА СЕРВЕРОВ:
Закрой оба окна (синее и желтое) или нажми Ctrl+C в каждом
